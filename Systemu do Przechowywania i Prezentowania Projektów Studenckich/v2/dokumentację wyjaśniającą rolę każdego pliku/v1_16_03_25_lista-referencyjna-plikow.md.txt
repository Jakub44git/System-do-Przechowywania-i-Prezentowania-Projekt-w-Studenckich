# Lista referencyjna plików projektu

Poniżej znajduje się lista wszystkich plików projektu z ich pełnymi ścieżkami, pogrupowana według modułów. Możesz wykorzystać tę listę jako referencję podczas umieszczania plików w odpowiednich katalogach projektu.

## Frontend (Vue.js + Tailwind CSS)

### Konfiguracja projektu
- `frontend/package.json` - zależności projektu
- `frontend/vite.config.js` - konfiguracja Vite
- `frontend/tailwind.config.js` - konfiguracja Tailwind CSS
- `frontend/.env` - zmienne środowiskowe
- `frontend/index.html` - główny plik HTML
- `frontend/tsconfig.json` - konfiguracja TypeScript (jeśli używasz TS)

### Struktura aplikacji
- `frontend/src/main.js` - punkt wejścia aplikacji
- `frontend/src/App.vue` - główny komponent
- `frontend/src/router/index.js` - konfiguracja routingu
- `frontend/src/store/index.js` - główny plik Pinia store

### Komponenty
- `frontend/src/components/layout/Navbar.vue` - górny pasek nawigacyjny
- `frontend/src/components/layout/Footer.vue` - stopka
- `frontend/src/components/layout/Sidebar.vue` - menu boczne (jeśli używane)

- `frontend/src/components/common/Button.vue` - komponent przycisku
- `frontend/src/components/common/Card.vue` - komponent karty
- `frontend/src/components/common/Alert.vue` - komponent alertu
- `frontend/src/components/common/Modal.vue` - komponent modalu
- `frontend/src/components/common/Pagination.vue` - komponent paginacji
- `frontend/src/components/common/Search.vue` - wyszukiwarka

- `frontend/src/components/projects/ProjectList.vue` - lista projektów
- `frontend/src/components/projects/ProjectCard.vue` - karta projektu
- `frontend/src/components/projects/ProjectForm.vue` - formularz projektu
- `frontend/src/components/projects/ProjectFilter.vue` - filtrowanie projektów

- `frontend/src/components/users/UserCard.vue` - karta użytkownika
- `frontend/src/components/users/UserList.vue` - lista użytkowników
- `frontend/src/components/users/LoginForm.vue` - formularz logowania

- `frontend/src/components/admin/AdminSidebar.vue` - menu boczne dla admina
- `frontend/src/components/admin/UserTable.vue` - tabela użytkowników
- `frontend/src/components/admin/CategoryManager.vue` - zarządzanie kategoriami

### Widoki (strony)
- `frontend/src/views/HomeView.vue` - strona główna
- `frontend/src/views/LoginView.vue` - strona logowania
- `frontend/src/views/ProjectsView.vue` - lista projektów
- `frontend/src/views/ProjectDetailsView.vue` - szczegóły projektu
- `frontend/src/views/UserDashboardView.vue` - panel użytkownika
- `frontend/src/views/TeacherDashboardView.vue` - panel nauczyciela
- `frontend/src/views/AdminDashboardView.vue` - panel administratora
- `frontend/src/views/NotFoundView.vue` - strona 404

### Store (Pinia)
- `frontend/src/store/modules/auth.js` - store autoryzacji
- `frontend/src/store/modules/projects.js` - store projektów
- `frontend/src/store/modules/users.js` - store użytkowników
- `frontend/src/store/modules/categories.js` - store kategorii

### Serwisy
- `frontend/src/services/api.js` - główny serwis API
- `frontend/src/services/authService.js` - serwis autoryzacji
- `frontend/src/services/projectService.js` - serwis projektów
- `frontend/src/services/userService.js` - serwis użytkowników
- `frontend/src/services/categoryService.js` - serwis kategorii

### Pomocnicze
- `frontend/src/utils/validators.js` - funkcje walidacji
- `frontend/src/utils/formatters.js` - formatowanie danych
- `frontend/src/utils/helpers.js` - pomocnicze funkcje
